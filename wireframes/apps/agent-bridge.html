② /api/ai/* — Claude MQ

AI Job Queue

routes/ai.ts
POST/api/ai/run

body: {prompt, metadata?}
→ {jobId, status:"queued", filePath, submittedAt}

GET/api/ai/status/:jobId

→ {jobId, status, result?, completedAt?}

Flow

1. POST /api/ai/run → submitTask() → .claude-mq/inbox/<jobId>.md
2. mqWatcher → 파일 감지 → status.json: processing
3. 외부 Claude CLI → inbox 읽기 → outbox 쓰기
4. GET status → outbox 결과 반환
