macOS / Windows Renderer = web (localhost:3000)
역할: Electron 데스크탑 클라이언트. Main process가 IPC + Tray + System integration 제공. Renderer는 별도 페이지가 아니라 WEB_APP_URL (기본 localhost:3000)에서 apps/web을 로드. UI는 web과 동일 — desktop-specific 화면 없음.
