M5 리터치 모듈 — Retouch

미설치 (예시) PBC: image-engine RETOUCH
대상: 사진관, 이커머스 상품 사진. 차별점: 콘텐츠 모듈과 별도 — RETOUCH 모드 + PRO_MODE_SYSTEM_PROMPT에 집중한 좁은 도메인. 왜 별도 모듈?: 콘텐츠 모듈은 너무 광범위. 리터치만 필요한 사용자는 더 저렴하게 + 워크플로우 최적화. 가격: ₩39,000 / 월 + 크레딧.

① 모듈 메타데이터

module.config.ts (예정)

{
  id: "retouch",
  label: "리터치",
  icon: "✨",
  pbc: ["pbc-image-engine"],   // RETOUCH 모드 + retouch-pro/free preset만
  nav: [
    { id: "editor", label: "에디터", route: "/retouch/editor", perm: "retouch:write" },
    { id: "batch", label: "배치", route: "/retouch/batch", perm: "retouch:write" },
    { id: "presets", label: "프리셋", route: "/retouch/presets", perm: "retouch:read" },
    { id: "projects", label: "프로젝트", route: "/retouch/projects", perm: "retouch:read" },
    { id: "history", label: "히스토리", route: "/retouch/history", perm: "retouch:read" }
  ],
  prismaModels: ["RetouchProject", "RetouchImage", "RetouchPreset"]
}
