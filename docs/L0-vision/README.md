# AXLE Vision

## 미션
컨설팅 업무의 반복적/수동적 프로세스를 AI로 자동화하여, 컨설턴트가 고부가가치 업무에 집중하도록 한다.

## 목표
- 사업계획서 작성 시간 50% 단축
- 서류 수집 자동화율 80%
- 지원사업 매칭 자동 추천
- 세션 간 학습으로 AI 품질 지속 향상

## 사용자
- **Primary**: FlowCoder + 여유솔루션 컨설팅 팀 (2~5명)
- **Secondary**: 고객사 담당자 (서류 업로드, 연구일지 작성)
- **Future**: SaaS 확장 (외부 컨설턴트)

## 핵심 가치
| Before | After (AXLE) |
|--------|-------------|
| 카톡/메일/폴더에 분산된 정보 | 고객사별 단일 대시보드 |
| 지원사업 수동 검색 | AI 크롤링 + 자동 매칭 |
| 사업계획서 수동 작성 | 이중 엔진 AI 자동 생성 |
| 서류 수집 카톡 요청 | 토큰 링크 → 업로드 → 자동 확인 |
| 인증 서류 수동 발급 | Desktop 자동 발급 |
| 미팅 메모 수기 | 녹음 → 전사 → 요약 → 메일 |
| 연구일지 엑셀 | AI 초안 + 웹 작성 |
