# AXLE 사용자 매뉴얼

AXLE는 정부 지원사업·벤처/연구소 인증·특허·재무 컨설팅 업무를 자동화하는 플랫폼입니다. 이 매뉴얼은 **컨설턴트가 AXLE를 사용하는 방법**을 설명합니다.

## 독자 대상

- 컨설팅 실무자 (고객사 관리, 사업계획서 작성, 미팅 전사 등)
- 조직 구성원으로 초대받은 팀원
- 조직 관리자 (일부 관리 기능 포함)

> 시스템 운영자 / 개발자 문서는 `docs/specs/`, `docs/plans/`를 참고하세요.

## 스크린샷

각 챕터의 이미지는 `./images/{chapter}/` 경로에 저장됩니다. 초기 배포 시점에는 placeholder 자리만 비어 있으며, 실제 캡처는 Playwright 자동화 스크립트(`e2e/manual-screenshots.spec.ts`)로 주입합니다.

```bash
# 스크린샷 재생성
npx playwright test e2e/manual-screenshots.spec.ts
```

## PDF 렌더링

이미지 주입이 완료되면 다음 명령으로 PDF를 생성합니다.

```bash
# markdown → PDF (pandoc + wkhtmltopdf 필요)
pandoc docs/manual/user/*.md \
  -o dist/axle-user-manual.pdf \
  --pdf-engine=wkhtmltopdf \
  --toc --toc-depth=2 \
  -V mainfont="Noto Sans KR" \
  -V geometry:margin=1in
```
