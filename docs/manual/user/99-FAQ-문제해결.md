# 99. FAQ · 문제해결

자주 묻는 질문과 흔한 문제 해결법을 모았습니다. 여기서 해답을 찾지 못하면 **[지원 채널](#지원-채널)**로 문의하세요.

---

## 1. 계정·로그인

### Q. 로그인 매직 링크가 오지 않아요.

1. **스팸함 확인** — `no-reply@axleai.io`에서 발송됩니다.
2. 수신 차단 리스트에서 제외.
3. 5분 후에도 오지 않으면 재요청.
4. 사내 메일 서버가 외부 메일을 차단했을 수 있습니다. 관리자에게 `axleai.io` 도메인 화이트리스트 요청.

### Q. 비밀번호가 없는데 어떻게 로그인하나요?

AXLE는 **비밀번호 없는(passwordless) 로그인**만 제공합니다. 매직 링크 또는 Google OAuth 중 하나를 사용하세요.

### Q. 조직이 여러 개인데 늘 잘못된 조직으로 들어가요.

마지막으로 선택한 조직이 기본값입니다. 사용자 메뉴 → **[조직 전환]**으로 바꾼 뒤 해당 조직 내에서 한 번이라도 작업하면 기본값이 갱신됩니다.

### Q. 초대받은 조직에 참여가 안 돼요.

- 초대 이메일 유효기간은 **7일**입니다. 만료되면 조직 관리자에게 재발송 요청.
- 본인 이메일과 초대 이메일이 다르면 초대 수락 시 오류가 납니다. 동일한 이메일로 로그인하세요.

## 2. 서류·업로드

### Q. 파일 업로드가 실패해요.

| 원인 | 해결 |
|------|------|
| 용량 초과 (50MB) | 분할 업로드 또는 압축 |
| 지원하지 않는 확장자 | PDF / DOCX / XLSX / HWP / 이미지만 가능 |
| 네트워크 불안정 | 재시도, 안정된 네트워크 이용 |
| 저장소 용량 부족 | 관리자에게 문의 (조직 플랜 확인) |

### Q. OCR 결과가 엉망이에요.

- 스캔 품질을 확인하세요 (저해상도, 기울어짐, 흐릿함)
- 재스캔하여 재업로드
- 심한 경우 수동 입력 후 OCR 결과와 병합

### Q. 삭제한 서류를 복구할 수 있나요?

**30일 이내**는 휴지통에서 복원 가능합니다. 서류 목록 상단 **[휴지통]** → 해당 파일 → **[복원]**.

## 3. 미팅·전사

### Q. 전사가 시작되지 않아요.

- 음원 파일 형식이 지원되는지 확인(MP3/M4A/WAV/WEBM)
- 파일이 손상되지 않았는지 확인
- 큐가 혼잡할 경우 몇 분 지연될 수 있습니다
- 30분 이상 경과 시 **[재시도]** 버튼으로 다시 실행

### Q. 전사 결과에 잘못된 단어가 많아요.

- 회사명·전문용어 등은 AI가 학습하지 못한 고유명사일 수 있습니다
- 결과를 **[편집]**으로 직접 수정 후 **[요약 재생성]**
- 발언자 구분이 필요하면 업로드 시 *화자 수*를 지정하세요

### Q. 녹음 파일이 너무 커요.

- 최대 200MB 제한
- 긴 미팅은 2~3개로 분할하거나, 저비트레이트(64kbps) m4a로 변환

## 4. AI·사업계획서

### Q. 사업계획서 초안 품질이 낮아요.

체크리스트:

- [ ] 고객사 마스터 프로필이 생성되어 있나요?
- [ ] 관련 서류가 **5개 이상** 업로드되어 있나요?
- [ ] 서류들의 OCR이 완료되었나요?
- [ ] 최근 미팅 전사가 있나요?
- [ ] 생성 옵션에서 *대상 지원사업*을 지정했나요?

### Q. AI 작업이 너무 오래 걸려요.

- 일반적으로 2~5분 소요
- 5분 이상: 큐 지연 가능성, 알림을 켜두고 다른 작업 진행
- 15분 이상: **[취소 후 재시도]** 또는 관리자에게 문의

### Q. AI가 이상한 내용을 생성했어요.

- 생성 결과는 **참고 자료**이며 반드시 검토 후 사용
- 섹션별 **[재생성]**으로 다른 결과를 받아볼 수 있습니다
- 심각한 오류는 해당 결과 화면의 **[문제 신고]**로 피드백

## 5. 알림·이메일

### Q. 알림이 오지 않아요.

1. `/settings/notifications`에서 해당 타입이 ON인지 확인
2. 웹 Push: 브라우저 알림 권한 확인
3. 이메일: 스팸함 확인, 수신거부 상태 확인
4. Telegram: 봇 차단 여부 확인

### Q. 알림이 너무 많아요.

- 타입별로 채널을 세분화하세요 (예: 자잘한 업데이트는 이메일, 긴급만 Push)
- 가장 자주 오는 알림 2~3개를 파악해 먼저 줄이세요

### Q. 이메일이 "스팸"으로 분류돼요.

수신자 측에서 `no-reply@axleai.io`를 연락처/화이트리스트에 추가해달라고 요청하세요.

## 6. 성능·접속

### Q. 화면이 느려요.

- 브라우저 캐시 삭제 후 재접속
- 조직에 고객사·프로젝트가 매우 많으면 목록 필터를 적용해 표시 건수를 줄이세요
- 오래된 브라우저는 최신 버전(Chrome/Safari/Edge)으로 업데이트

### Q. 특정 페이지에서 오류가 납니다.

1. **새로고침** (⌘+R / Ctrl+R)
2. 다른 페이지로 이동 후 재접속
3. 계속되면 오른쪽 하단 **[오류 신고]** 버튼 사용

### Q. 모바일에서 일부 기능이 안 보여요.

일부 복잡한 편집 기능(양식 정밀 편집 등)은 데스크톱에 최적화되어 있습니다. 모바일은 조회·승인·간단 업로드에 적합합니다.

## 7. 데이터·보안

### Q. 우리 데이터는 어디에 저장되나요?

- 구조화 데이터: Supabase PostgreSQL
- 파일: Supabase Storage
- 백업: 일일 자동 백업, 7일 보존

### Q. AI가 우리 데이터로 학습하나요?

- AXLE 내부 AI(SkillPattern)는 **조직 내에서만** 학습에 사용됩니다
- 외부 API(Gemini 등)는 학습에 사용되지 않도록 옵션이 설정되어 있습니다
- 자세한 내용은 서비스 이용약관 참고

### Q. 계정 탈퇴 시 데이터는?

개인 탈퇴 시 개인 식별 정보는 30일 내 삭제되지만, 조직이 가진 프로젝트·서류 데이터는 조직 소유로 남습니다. 조직 자체를 삭제하려면 관리자에게 요청하세요.
